Proceso de Registro



Nombre y Apellidos

  • El nombre y los apellidos son campos obligatorios y no pueden dejarse vacíos durante el proceso de registro.


    • Correo Electrónico

  • Se requiere un correo electrónico válido para el registro.
  • El correo electrónico debe contener el símbolo "@" para ser considerado válido.

    • Documento de Identidad

  • El DNI es un campo obligatorio y debe tener entre 9 y 10 caracteres.
  • Además de su función principal de identificación, el DNI también se utiliza para verificar si el usuario tiene algún cupón.

    • Comentarios Opcionales

  • Los usuarios tienen la opción de dejar comentarios durante el registro, proporcionando una vía para expresar preferencias o solicitudes especiales.

    • Confirmación de Mayoria de Edad

  • Los usuarios deben afirmar que tienen la mayoría de edad para poder completar el registro.
  • Esta confirmación es esencial para cumplir con los requisitos legales y de privacidad.




    • Verificación de Cupón

  • El DNI proporcionado se utiliza para verificar si el usuario tiene algún cupón asociado.
  • Si el usuario tiene un cupón válido, este se puede aplicar durante el proceso de reserva.
